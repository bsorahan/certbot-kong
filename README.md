# Certbot Kong

## Install
To 

```
cd certbot-kong
pip install .
```

## Run
```
DOMAINS=example.com,*.test.com,abc.example.com
EMAIL=foo@example.com
KONG_ADMIN_URL=http://localhost:8001
certbot install -i certbot-kong:kong
```
